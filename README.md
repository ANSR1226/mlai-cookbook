## 🏃‍♂️ How to Use This Repo

1. **Find your topic:** Head down to the [Navigation Map](#️-navigation-map) and click on the area you want to explore.
2. **Dive into the code:** Most folders contain documented Jupyter Notebooks (`.ipynb`) or clean Python scripts (`.py`).
3. **Run it yourself:** Clone the repo, spin up your environment, and start breaking things (that's how we learn!).
